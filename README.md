# Cemetery Management System (MCMS) - Backend

## Overview
Node.js/Express backend for the Cemetery Management System.

## deployed URL
Production API: `https://cemetry360-backend.azurewebsites.net/api`

## Setup
1. `npm install`
2. Configure `.env` (see `.env.example`)
3. `npm start`
